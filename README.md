# Motor Ride Tracker

Web-based motorcycle ride tracking system developed using Laravel and MySQL for recording rides, fuel consumption, maintenance history, and ride statistics.

## Features
- Ride history management
- Distance tracking
- Fuel consumption records
- Motorcycle maintenance logs
- Dashboard statistics
- User account management

## Technologies
- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Database
The system uses MySQL database to store:
- User data
- Motorcycle information
- Ride history
- Fuel records
- Maintenance records

## System Functions
- Add ride records
- Track travel distance
- Monitor fuel usage
- Record maintenance activities
- Display ride statistics
- Manage motorcycle data

## Modules
### User Management
- Registration
- Login
- Profile Management

### Ride Tracking
- Start point
- Destination
- Distance
- Travel date

### Fuel Management
- Fuel purchase records
- Fuel cost tracking
- Fuel efficiency calculation

### Maintenance Management
- Service history
- Oil change records
- Spare part replacement records
